## Package

If you are looking for the Starlight plugin package, you can find it in the [`packages/starlight-telescope/`](/packages/starlight-telescope/) directory.

## Project structure

This project uses pnpm workspaces to develop a single Starlight plugin from the `packages/starlight-telescope/` directory. A Starlight documentation site is also available in the `docs/` directory that is also used for testing and demonstrating the Starlight plugin.
