## CSS Custom Properties

starlight-telescope uses CSS custom properties for all colors, making it easy to match your site's theme.

### Available Properties

The plugin automatically inherits Starlight's design tokens. You can override any of these:

```css
:root {
  /* Overlay/backdrop */
  --telescope-overlay-bg: var(--sl-color-backdrop-overlay);

  /* Modal backgrounds */
  --telescope-modal-bg: var(--sl-color-bg-nav);
  --telescope-modal-bg-alt: var(--sl-color-bg);

  /* Accent colors */
  --telescope-accent: var(--sl-color-accent);
  --telescope-accent-low: var(--sl-color-accent-low);
  --telescope-accent-high: var(--sl-color-accent-high);
  --telescope-accent-hover: color-mix(in srgb, var(--telescope-accent) 10%, transparent);
  --telescope-accent-selected: color-mix(in srgb, var(--telescope-accent) 18%, transparent);

  /* Text colors */
  --telescope-text-primary: var(--sl-color-white);
  --telescope-text-secondary: var(--sl-color-gray-3);

  /* Borders */
  --telescope-border: var(--sl-color-hairline-light);
  --telescope-border-active: var(--sl-color-gray-5);

  /* Feature colors */
  --telescope-pin-color: var(--sl-color-purple);
  --telescope-tag-color: var(--sl-color-green);
}
```

## Light and Dark Mode

The plugin inherits Starlight's light/dark mode automatically via CSS variables. Starlight handles `prefers-color-scheme` detection, and telescope inherits those values.

To customize colors per mode:

```css
:root {
  --telescope-accent: #7c3aed;
}

:root[data-theme='light'] {
  --telescope-accent: #6d28d9;
  --telescope-modal-bg: #ffffff;
}
```

## Configuration-Based Theming

Set theme colors directly in your config:

```js
starlightTelescope({
  theme: {
    accentColor: '#7c3aed',
    accentHover: 'rgba(124, 58, 237, 0.1)',
    accentSelected: 'rgba(124, 58, 237, 0.15)',
    pinColor: '#f59e0b',
    modalBackground: '#1e1e2e',
  }
})
```

## CSS Override Method

For more control, override CSS variables in your custom stylesheet:

```css
/* src/styles/custom.css */
:root {
  --telescope-accent: var(--sl-color-accent);
  --telescope-modal-bg: var(--sl-color-bg);
}
```

## Accessibility

When customizing colors, ensure sufficient contrast:
- **Text on backgrounds:** Minimum 4.5:1 contrast ratio (WCAG AA)
- **Interactive elements:** Clearly distinguishable focus states
- **Highlight colors:** Visible against both modal background and text

Use tools like [WebAIM Contrast Checker](https://webaim.org/resources/contrastchecker/) to verify your color choices.

## Customizing Specific Elements

### Search Input

```css
.telescope__search-input {
  font-size: 20px;
  font-family: var(--__sl-font);
}
```

### Result Items

```css
.telescope__result-item {
  padding: 16px 24px;
}

.telescope__result-title {
  font-size: 16px;
}

.telescope__result-description {
  font-size: 13px;
}
```

### Match Highlighting

```css
.telescope__highlight {
  background-color: rgba(255, 200, 0, 0.3);
  border-radius: 3px;
}
```

### Pin Button

```css
.telescope__pin-button {
  color: #f59e0b;
}

.telescope__pin-button--pinned {
  color: #f59e0b;
  background-color: rgba(245, 158, 11, 0.2);
}
```

## Available CSS Classes

The plugin uses BEM naming convention (`telescope__element--modifier`):

| Class | Element |
|-------|---------|
| `.telescope` | Modal dialog container |
| `.telescope__modal` | Modal content wrapper |
| `.telescope__search-input` | Search input field |
| `.telescope__tabs` | Tab bar container |
| `.telescope__tab` | Individual tab |
| `.telescope__tab--active` | Active tab |
| `.telescope__result-item` | Search result item |
| `.telescope__result-item--selected` | Selected result item |
| `.telescope__result-title` | Result title |
| `.telescope__result-description` | Result description |
| `.telescope__pin-button` | Pin/bookmark button |
| `.telescope__pin-button--pinned` | Pinned state |
| `.telescope__highlight` | Match highlight |
| `.telescope__section-separator` | Section header |
| `.telescope__footer` | Modal footer |

## Reduced Motion

The plugin respects `prefers-reduced-motion`. When users prefer reduced motion, all animations are disabled:

```css
@media (prefers-reduced-motion: reduce) {
  .telescope[open],
  .telescope__result-item {
    animation: none;
    transition: none;
  }
}
```

## Responsive Design

The modal is responsive by default:
- **Desktop:** 640px width, centered
- **Tablet/Mobile:** 95% width with adjusted spacing

Custom breakpoint adjustments:

```css
@media (max-width: 768px) {
  .telescope__modal {
    border-radius: 0;
  }

  .telescope__search-input {
    font-size: 16px;
  }
}
```
